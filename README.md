#Personal Blog Layout

Um projeto de interface de blog moderno e limpo, focado em legibilidade, organização de conteúdo e boas práticas de estruturação HTML/CSS. Desenvolvido para praticar o uso de colunas (Sidebar) e componentes de navegação.

**Acesse o projeto ao vivo:  https://gugu494.github.io/personal-blog-layout/
Tecnologias
Este projeto foi construído utilizando:
* **HTML5 Semântico** - Uso de tags como `<header>`, `<main>`, `<aside>` e `<footer>`.
* **CSS3** - Layout baseado em Flexbox/Grid, estilização de estados de hover e responsividade.
* **Google Fonts** - Integração de fontes personalizadas para melhor UI.

#Funcionalidades e Diferenciais
* Sidebar Completa: Área lateral com campo de busca, lista de categorias e nuvem de tags.
* Cards de Postagem: Layout estruturado para imagens de destaque com título, autor e resumo.
* Componentes de UI: Botões de tags estilizados e lista de categorias interativa.
* Clean Design: Foco na hierarquia visual, facilitando a leitura do usuário.

O que foi praticado:
1. Posicionamento de Elementos:** Uso de `display: flex` para alinhar os elementos da sidebar e do conteúdo principal.
2. Responsividade: Estrutura adaptável para diferentes tamanhos de tela.
3. Organização de Arquivos: Separação clara entre estrutura (HTML), estilo (CSS) e ativos (Images).
